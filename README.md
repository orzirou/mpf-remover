# MpfRemover

## Summary

このアプリは Google Photos の設定を節約画質(旧設定名称 高画質)にしているにも関わらず、圧縮保存されない画像が圧縮されるするためのものです。

EXIF データ内に MPF という情報が含まれる場合、Google Photos では圧縮されないため MPF タグを削除します。  
※exiftool を使ったコマンド操作が難しい方向けのアプリです。
